# Agentic RAG System

An Agentic Retrieval-Augmented Generation (RAG) system where an intelligent agent
decides whether document retrieval is required before answering a query.

## Features
- PDF-based knowledge ingestion
- Intelligent query routing (Search vs Direct)
- Local vector store (ChromaDB)
- Flask API backend
- Deployed on Render

## Run Locally
pip install -r requirements.txt
python app.py
